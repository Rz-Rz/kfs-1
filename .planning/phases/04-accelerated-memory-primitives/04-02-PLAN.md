# Phase 4 Plan 02

Implement the first real accelerated `memcpy` backend behind the canonical memory facade.

Must-haves:
- `memcpy` keeps the scalar contract for zero length, return value, ordinary unaligned pointers, and surrounding bytes.
- `SSE2` selection happens only through `dispatch.rs`.
- Boot and host tests can distinguish `SSE2` selection from scalar fallback.
